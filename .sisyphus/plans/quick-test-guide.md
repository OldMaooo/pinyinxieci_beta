# 快速测试指南

## 应用地址
http://localhost:3009

---

## 1️⃣ Setup 页面测试（立即测试）

### WEAK 词显示
- [ ] 看是否有词组下面有**红色实线**
- [ ] 红色实线表示该词组中有 WEAK 词

### MASTERED 词显示  
- [ ] 看是否有词组下面有**绿色实线**
- [ ] 绿色实线表示该词组中所有词都是 MASTERED

---

## 2️⃣ 练习页面测试

### WEAK 词颜色
1. 选择任意单元，点击"开始练习"
2. [ ] WEAK 词的**拼音**显示淡红色（`text-red-300`）
3. [ ] WEAK 词的**汉字**显示淡红色（`text-red-300`）

### 短期标记
1. [ ] 标记一个错题（点击红色框）
2. [ ] 标记一个正确题（点击绿色框）
3. [ ] 框框颜色正确显示

---

## 3️⃣ "仅错题"筛选测试

### 筛选功能
1. 返回 Setup 页
2. [ ] 勾选"仅错题"复选框
3. [ ] 只显示 WEAK 状态的词组
4. [ ] WEAK 词组有红色下划线
5. [ ] MASTERED 词组有绿色下划线
6. [ ] NEW 词组无下划线

---

## 4️⃣ 新的一天清空测试

### 进入新的一天
1. [ ] 关闭应用，修改系统日期为明天
2. [ ] 重新打开应用
3. [ ] 进入练习
4. [ ] 所有词的短期标记都是白色（未标记状态）

---

## 5️⃣ 闪卡模式测试

### WEAK 词颜色
1. 点击 Setup 页右上角的闪卡图标（眼睛图标）
2. [ ] WEAK 词显示淡红色（`text-red-300`）
3. [ ] 与练习页面的颜色一致

### 本轮错题显示
1. [ ] 本轮标记的错题显示纯红色（`text-red-500`）
2. [ ] 纯颜色比 WEAK 状态的淡红色更深

---

## 6️⃣ 存档功能测试

### 完成一轮练习
1. 进入练习，完成自由练习、自测、家长终测
2. [ ] 点击"存档并结束"按钮

### 数据更新
1. 打开浏览器控制台（F12 → Console）
2. [ ] 运行：`console.log(window.mastery['3up-单元1-山坡-0'])`
3. [ ] 查看返回的对象包含：
   - `consecutive_green` (数字)
   - `last_practice_date` (日期字符串 YYYY-MM-DD)
