# MD中间数据转JSON词库工作计划

## 目标
将 `AI_Data_Generation_Kit/intermediate_data/二年级上册.md` 转换为JSON词库格式，并更新到 `dist/data/二年级上册.json`

## 用户选择：存储原始课名
应用显示时使用MD中的原始课名（如"阅读1"、"识字2"、"语文园地一"），而不是统一为"单元X"。

## JSON格式（新增lessonName字段）
```json
{
  "version": "1.0",
  "buildDate": "ISO日期",
  "gradeSemester": "二年级上册",
  "count": 总词数,
  "wordBank": [
    {
      "word": "词语",
      "pinyin": "",
      "grade": "二年级",
      "semester": "上册",
      "unit": 数字,
      "lessonName": "阅读1"
    }
  ]
}
```
