# 词库处理 - 一年级上册

## 目标
从PDF中提取词语表和写字表，生成正确的词库JSON

## 步骤

### Step 1: 分析PDF结构
- 读取PDF文件
- 定位"词语表"和"写字表"的位置
- 确定提取格式

### Step 2: 提取词语表
- 提取每个单元的词语表内容
- 解析词语和拼音

### Step 3: 提取写字表
- 提取每个单元的写字表内容（单字）
- 标记为识字

### Step 4: 数据处理
- 合并词语表和写字表
- 去重处理（写字表中的字如果已在词语表中，去掉）
- 将剩余的单字进行组词

### Step 5: 格式转换
- 转换为目标JSON格式
- 单元命名规范（数字单元直接写数字，语文园地按"语文园地+序号"）

### Step 6: 验证
- 验证生成的JSON格式
- 检查数据完整性

## 输入/输出

**输入**: `dist/data/【人教版】一年级上册(2024秋版)语文电子课本.pdf`

**输出**: `public/data/一年级上册.json`

## 数据格式

```json
{
  "version": "1.0",
  "buildDate": "2026-02-07T...",
  "gradeSemester": "一年级上册",
  "count": N,
  "wordBank": [
    {
      "word": "山坡",
      "pinyin": "shān pō",
      "grade": "一年级",
      "semester": "上册",
      "unit": 1
    }
  ]
}
```
